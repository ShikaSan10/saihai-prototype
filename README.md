# SAIHAI（采配）- 建設業向けAI配置最適化システム

建設現場の職人配置と車両管理を最適化するAIシステムのプロトタイプです。

## 🎯 主な機能

### 配置表
- **時間帯別カラム表示**（午前/午後/終日）
- **1人の職人が複数現場に対応**（1日最大3現場程度）
- **ドラッグ&ドロップ**で手動配置調整
- **自動配置機能** - 優先度・スキル・移動時間を考慮したAI最適化

### タイムライン
- 全作業員の1日のスケジュールを可視化
- 現場間の移動時間を表示
- 非稼働者のグレーアウト表示

### 作業員マスタ
- 保有資格・免許の管理
- スキル情報
- **稼働状況管理**
  - 休暇（夏季休暇など）
  - 怪我・病気
  - 一時帰国（技能実習生）
- 復帰予定日の設定

### 車両管理
- 車両の稼働状況管理
  - 故障
  - 車検
  - 定期点検
- 復帰予定日の設定
- 使用状況の可視化

## 🚀 起動方法

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ブラウザで http://localhost:5173 を開く
```
